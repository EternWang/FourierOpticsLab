# Data

## `raw/`
Hand-entered measurements and constants exactly as recorded in the lab notebook.

Files:

- `screen_angle.csv`
  Screen distance `L` and order separation `y` for the screen-angle method.
- `camera_calibration.csv`
  Camera calibration spacing and pixel counts.
- `slit_cutoff.csv`
  Variable slit cutoff width and instrument constants.
- `abbe_params.csv`
  Parameters used to estimate numerical aperture and the Abbe resolution limit.

## `processed/`
Generated by `analysis/analyze.py` and intended to be reproducible rather than edited by hand.

- `results.json`: full structured output
- `grating_results.csv`: summary table for Experiment 1
